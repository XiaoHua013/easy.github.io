### 1.0.4
**2021年8月1號:**
1. **新增** 強化秘藥

#### 1.0.4更新小助手
- 配置文件對比1.0.2版本有變動,請添加以下到config.yml中或者刪除舊config讓重新生成
```
# 強化秘藥 增加的強化概率 加算
secretMedicines: 4
# 強化秘藥 可持續時間, 單位 秒
secretMedicinesTime: 1800
```
- 語言文件對比1.0.2版本有變動,zh_CN.yml中或者刪除舊的讓重新生成
```
# 強化秘藥
secretMedicines:
  name: "&e強化秘藥"
  lore:
    - ''
    - '&f[ &c使用效果 &f]'
    - ''
    - '&f>> 使用後,可以提升強化成功率'
    - '&f>> 效果可持續30分鐘。'
```
