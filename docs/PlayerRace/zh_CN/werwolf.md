### 种族介绍:
```
一种在夜晚拥有强大力量的黑暗生物
```

### 种族来源:
```
- 在夜黑风高之时,人类被狼击杀,将会转换成狼人
```

### 种族天赋:
```
1. 晚上击杀生物会恢复能量值
2. 晚上吃 ** 生肉 ** 会恢复能量值
3. 会随着时间来极慢恢复能量值
```

### 被动技能:
```
2. 晚上会快速回血
3. 晚上近战攻击力提升
4. 晚上远程攻击力大幅度提升
6. 晚上掉落减伤
```

### 主动技能:
```
1. 召唤狼: 手持生猪肉左击 消耗一定能量值,可以召唤狼
2. 猛冲: 手持羽毛左击 消耗一定能量值,可以向前冲刺一段距离
```

### 种族弱点
```
1. 狼人害怕阳光,在白天会失去大部分能力
```
