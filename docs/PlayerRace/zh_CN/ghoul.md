### 种族介绍:
```
一种自愿放弃做人,拥有诡异力量的邪恶生物
```

### 种族来源:
```
- 人类被猪人击杀转换而来
- 人类被食尸鬼使用 邪恶诅咒 转换而来
```

### 种族天赋:
```
 1. 会随着时间正常恢复能量值
```

### 被动技能:
```
1. 与僵尸休战
2. 近战吸血(消耗能量)
3. 近战 消耗敌人能量
```

### 主动技能:
```
1. 召唤猪人: 手持金块左击 消耗一定能量值,可以召唤出极度贪婪的猪人
2. 邪恶诅咒: 手持骨头打击人类玩家  消耗一定能量值,可以诅咒该人类玩家一段时间后被迫成为食尸鬼
```

### 种族弱点
```
1. 怕水
```
