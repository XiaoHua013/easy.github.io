### 種族介紹:
```
一種自願放棄做人,擁有詭異力量的邪惡生物
```

### 種族來源:
```
- 人類被豬人擊殺轉換而來
- 人類被食屍鬼使用 邪惡詛咒 轉換而來
```

### 種族天賦:
```
 1. 會隨著時間正常恢復能量值
```

### 被動技能:
```
1. 與僵屍休戰
2. 近戰吸血(消耗能量)
3. 近戰 消耗敵人能量
```

### 主動技能:
```
1. 召喚豬人: 手持金塊左擊 消耗一定能量值,可以召喚出極度貪婪的豬人
2. 邪惡詛咒: 手持骨頭打擊人類玩家  消耗一定能量值,可以詛咒該人類玩家一段時間後被迫成為食屍鬼
```

### 種族弱點
```
1. 怕水
```
