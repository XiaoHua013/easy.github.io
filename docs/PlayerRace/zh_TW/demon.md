### 種族介紹:
```
一種通過神秘儀式,轉換為邪惡生物的種族
```

### 種族來源:
```
- 人類身穿鎖鏈套,在地獄葬身巖漿中轉換而來
```

### 種族天賦:
```
 1. 會隨著時間正常恢復能量值
 2. 在巖漿中快速恢復能量
```

### 被動技能:
```
1. 火焰免役
2. 巖漿免役
4. 在巖漿中快速恢復血量
```

### 主動技能:
```
1. 發射火焰彈 手持火焰彈左擊,消耗能量,直接發射
2. 生成蜘蛛網 手持墨囊左擊,消耗能量,生成蜘蛛網保護自己
```

### 種族弱點
```
1. 無法穿除了鎖鏈以外的裝備
```
