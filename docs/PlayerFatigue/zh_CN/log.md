### 1.1.3
**2020年8月31日**
1. **新增** 定时任务,可切换恢复在线还是离线玩家的配置

#### 1.1.3更新小助手
config.yml中添加了如下内容: 请自行添加,或者重新生成
```
# 定时恢复 是否开启离线恢复,false: 玩家在线才会恢复  true: 所有玩家都会恢复(包括离线)
timingOfflineRecovery: false
# 是否开启定时恢复模式,到了设定的时间会直接全部恢复体力,
# 根据上面的配置决定会恢复在线还是恢复离线
timingRecovery: false
# 定时恢复模式的恢复时间
timing: 00:00
```
