```
# 是否開啟緩存

```

### 1.1.3
**2020年8月31日**
1. **新增** 定時任務,可切換恢復在線還是離線玩家的配置

#### 1.1.3更新小助手
config.yml中添加了如下內容: 請自行添加,或者重新生成
```
# 定時恢復 是否開啟離線恢復,false: 玩家在線才會恢復  true: 所有玩家都會恢復(包括離線)
timingOfflineRecovery: false
# 是否開啟定時恢復模式,到了設定的時間會直接全部恢復體力,
# 根據上面的配置決定會恢復在線還是恢復離線
timingRecovery: false
# 定時恢復模式的恢復時間
timing: 00:00
```
